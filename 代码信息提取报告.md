# SUPEREV 营销分析助手 代码信息提取报告

## 0. 结论先看

这个项目当前是一个 **前后端同仓的单次营销诊断工具**：

- 前端：`React + TypeScript + Vite`
- 后端：`Node.js + Express`
- 分析引擎：本地规则计算 + 可选 AI 增强报告
- 当前已经打通：
  - 数据输入
  - 自动识别
  - 数据匹配校对
  - 结果页展示
  - PDF 导出
- 当前还没有打通：
  - 历史记录
  - 渠道级对比漏斗
  - 销售个人转化率
  - 真正的 ROI 分析
  - 持续趋势/预警体系

### 预算建议 `budgetRecommendations` 怎么生成？
答案：**纯规则生成**。  
代码位置：  
[shared/marketing-engine.ts](/Users/Zhuanz/Desktop/AI编程/superev-营销分析助手/shared/marketing-engine.ts#L1547) 第1547行

核心逻辑：
1. 计算每个产品：
   - `spendShare = spend / totalSpend`
   - `dealShare = deals / totalDeals`
   - `score = dealShare - spendShare`
2. `score` 高的产品视为更值得加预算
3. `score` 低的产品视为更应该减预算
4. 调整金额：
```ts
reallocateAmount = max(
  productSpendTotal * (overallRating === "🔴" ? 0.15 : 0.08),
  worseSpend * 0.12
)
```
